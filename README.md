# Iris Dataset 3-Class Neural Network

## Project Overview
This project implements a simple **3-class neural network from scratch using NumPy** to classify the famous Iris dataset. The network has:
- Input layer: 4 features
- Hidden layer: 10 neurons (ReLU activation)
- Output layer: 3 neurons (Softmax activation)

It demonstrates the basics of:
- Forward propagation
- Backpropagation
- Cross-entropy loss
- One-hot encoding
- Decision making for multi-class classification

---

## Dataset
**Iris dataset**:
- 150 samples, 4 features (sepal length, sepal width, petal length, petal width)
- 3 classes: Setosa (0), Versicolor (1), Virginica (2)

The dataset is split into **train (80%)** and **test (20%)** sets.
